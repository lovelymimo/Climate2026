📄 Climate Safety Hub – MVP 기획문서
1. 프로젝트 개요

**Climate Safety Hub(기후안전허브)**는
기후변화로 인해 증가하는 국지성 호우와 침수 위험을
시민이 직접 확인하고, 위험 징후를 공유함으로써
도시 차원의 기후안전 대응을 가능하게 하는 서비스이다.

본 MVP는
경기도 전역을 대상으로 침수 위험을 시각화하고,
시민 참여를 통해 위험 정보를 축적하는 최소 기능 버전이다.

2. MVP 목표

경기도 침수 위험 정보를 직관적인 지도 기반 UI로 제공한다.

시민이 자신의 거주 지역을 선택해 위험 여부를 확인할 수 있게 한다.

시민이 발견한 침수·배수 위험을 간단히 제보할 수 있도록 한다.

서비스의 핵심 가치(예방, 참여, 확장 가능성)를 빠르게 검증한다.

3. 핵심 사용자(User)
1) 경기도 거주 시민

장마철 또는 강우 예보 시,
내가 사는 지역이 침수 위험이 있는지 알고 싶은 사용자

위험해 보이는 지점을 쉽게 제보하고 싶은 사용자

4. MVP 핵심 유저 스토리 (User Stories)
US-1. 시민은 내가 사는 지역의 침수 위험을 확인하고 싶다

As a 경기도 거주 시민
I want to 경기도 전체 지도에서 시/구를 선택해
So that 과거 침수 위험이 있었는지 직관적으로 확인할 수 있다.

US-2. 시민은 위험해 보이는 지점을 간단히 제보하고 싶다

As a 시민 사용자
I want to 지도에서 위치를 선택하고 간단한 정보만 입력해
So that 침수나 배수 문제를 빠르게 공유할 수 있다.

US-3. 시민은 자신의 참여가 반영되고 있음을 느끼고 싶다

As a 제보한 시민
I want to 내가 남긴 제보가 지도에 표시되는 것을 보고
So that 서비스에 대한 신뢰와 지속적인 참여 동기를 얻는다.

8. MVP 성공 기준

본 MVP는 다음 조건을 충족할 경우 성공으로 판단한다.

사용자가 자신의 거주 지역을 선택해 침수 위험을 확인할 수 있다.

사용자가 위험 지점을 직접 제보할 수 있다.

제보 정보가 지도에 반영되어 참여가 시각적으로 확인된다.

서비스의 확장 가능성(데이터·기능·협력)이 명확히 드러난다.
