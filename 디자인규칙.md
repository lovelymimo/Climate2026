# Climate Safety Hub - 디자인 규칙

## 4. 로고 및 브랜드 컬러

### 로고 파일
- **파일 위치**: `logo.png`
- 모든 화면에서 일관되게 활용

### 주요 컬러 팔레트 (로고 기반)

| 색상명 | HEX 코드 | 용도 |
|--------|----------|------|
| **Primary Green** | `#2ECC71` | 주요 액션 버튼, 안전/긍정 표시, "Climate" |
| **Primary Blue** | `#2980B9` | 헤더, 링크, 정보 강조, "Safety Hub" |
| **Light Blue** | `#5DADE2` | 배경 하이라이트, 물/침수 관련 요소 |
| **White** | `#FFFFFF` | 배경, 카드, 텍스트 영역 |
| **Dark Gray** | `#2C3E50` | 본문 텍스트 |
| **Light Gray** | `#ECF0F1` | 섹션 구분 배경, 비활성 요소 |
| **Warning Red** | `#E74C3C` | 위험 알림, 높은 침수 위험 표시 |
| **Warning Orange** | `#F39C12` | 주의 알림, 중간 침수 위험 표시 |

### 그라데이션
- **Shield Gradient**: `#2ECC71` → `#2980B9` (로고 방패 그라데이션)
- 주요 헤더나 강조 영역에 활용 가능

## 5. UI/UX 디자인 원칙

### 직관성
- 한눈에 이해할 수 있는 아이콘과 레이블 사용
- 3클릭 이내로 주요 기능 접근 가능
- 명확한 시각적 계층 구조

### 깔끔함
- 충분한 여백(White Space) 확보
- 요소 간 일관된 간격 유지 (8px 기반 그리드)
- 불필요한 장식 요소 최소화

### 접근성
- 충분한 색상 대비 (WCAG 2.1 기준)
- 터치 타겟 최소 44px × 44px
- 명확한 포커스 표시

### 반응형
- 모바일 우선(Mobile First) 디자인
- 주요 브레이크포인트: 360px, 768px, 1024px

### 피드백
- 모든 사용자 액션에 즉각적인 시각적 피드백
- 로딩 상태 명확히 표시
- 성공/실패 메시지 직관적으로 전달
